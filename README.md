# math_mod

Рассматривается задача размещения авиационного хаба, обслуживающего два
географических региона:
- Регион А: страны Индостана;
- Регион В: государства Европы.
Каждый регион представлен набором ключевых аэропортов: nA = 11 аэропортов для Индостана и nB = 15 аэропортов для Европы.
Требуется определить пространственные координаты (xh, yh) хаба, обеспечивающие минимальное суммарное взвешенное расстояние перевозки пассажиров.
